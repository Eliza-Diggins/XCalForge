MyCustomLibrary#

class library.SKELETON.MyCustomLibrary(directory: str | Path)[source]#

ModificationLibrary for simulating Gaussian perturbations to ARF files.

Defines: - Allowed hyperparameters (mu: bias center, sigma: bias width) - Logic for generating distorted ARFs - XSPEC model construction - Fitting routines for temperature recovery analysis

Example cleanup: optionally remove temporary ARFs if generated per parameter.
